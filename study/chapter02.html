← 戻る

2-1 型の宣言と変換

数値と文字列の相互変換は、C++において基本中の基本です。

// int → string
int num = 123;
std::string str = std::to_string(num);

// string → int
std::string input = "456";
int val = std::stoi(input);

2-2 文字列操作

文字列の扱いには std::string を使います。C言語の char* とは異なり、より直感的に操作できます。

std::string name = "Taro";
name += " Yamada"; // 文字列の連結
std::cout << name << std::endl;

// 部分文字列の取得
std::string part = name.substr(0, 4);

// 1文字ずつ処理
for (char c : name) {
  std::cout << c << ' ';
}
