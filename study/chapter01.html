1-1 インクルードと名前空間

C++では必要な機能を使うためにヘッダーファイルを#includeで読み込みます。 標準の機能を省略して使うためにusing namespace std;を使いますが、 規模の大きいプロジェクトでは衝突を避けるため明示するのが望ましいです。

#include <iostream>
using namespace std;

int main() {
  cout << "Hello, C++!" << endl;
  return 0;
}

1-2 標準入出力

標準入力にはcin、出力にはcoutを使います。スペース区切りの入力が可能で、改行で終了します。

#include <iostream>
using namespace std;

int main() {
  int x;
  string name;
  cin >> x >> name;
  cout << "数値: " << x << ", 名前: " << name << endl;
  return 0;
}

1行すべてを取得したい場合はgetlineを使います。

string line;
getline(cin, line);
