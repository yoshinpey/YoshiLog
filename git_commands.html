Git 基本コマンドまとめ

🛠 初期設定

コマンド説明
git config --global user.name "名前"ユーザー名を設定
git config --global user.email "メールアドレス"メールアドレスを設定
git config --list現在の設定内容を確認

📁 リポジトリの作成・取得

コマンド説明
git initカレントディレクトリをGitリポジトリにする
git clone <URL>リモートリポジトリをローカルに複製

📦 ファイルの追加・コミット

コマンド説明
git status作業状況を確認
git add <ファイル>指定ファイルをステージに追加
git add .変更全体をステージに追加
git commit -m "メッセージ"ステージ済みの変更を記録
git rm <ファイル>ファイルを削除してステージに追加

🔍 履歴と差分

コマンド説明
git logコミット履歴を詳細表示
git log --oneline履歴を1行ずつ簡潔に表示
git diffステージ未追加の差分を確認
git diff --stagedステージ済みの差分を確認

🚀 リモートリポジトリ連携

コマンド説明
git remote -vリモートのURLを確認
git push origin mainmainブランチをリモートに送信
git pull origin mainリモートの変更を取得して統合
git fetchリモートの変更を取得するが統合しない

🌱 ブランチ管理

コマンド説明
git branchローカルのブランチ一覧を表示
git branch <名前>新しいブランチを作成
git checkout <名前>指定ブランチに切り替え
git checkout -b <名前>新規作成と切替を同時に実行
git merge <名前>指定ブランチを現在のブランチに統合

♻️ 巻き戻し・リセット

コマンド説明
git restore <ファイル>ステージしていない変更を取り消し
git reset <ファイル>ステージを解除
git reset --hard HEADすべての変更を取り消す（注意）

🔧 補助コマンド

  • git stash：作業途中の変更を一時退避
  • git stash pop：退避した変更を再適用
  • git tag <タグ名>：特定コミットにタグ付け
  • git revert <コミットID>：指定コミットを打ち消す新規コミットを作成
